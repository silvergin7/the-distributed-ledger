# Agila metoder - Grupparbete
Här är mallen för ert grupparbete.

## Att göra
- [ ] Gå in på ert repo och välj fliken `Settings`, klicka på `Pages` till vänster, och klicka sedan i dropdownen på `GitHub Actions` (istället för `Deployg from a branch`).
- [ ] Läs igenom avsnittet "Dokumentation" nedan, och bekanta er med mappens innehåll.
- [ ] Läs igenom avsnittet "Kod" nedan.
- [ ] Markera dessa uppgifter som klara genom att sätta ett kryss i rutan, såsom jag har gjort på nästa rad.
- [x] Klart! Nu kan ni börja jobba.

## Dokumentation
Se till att fylla i dokumentationen noga, då den agerar betygsunderlag i den här kursen.

I mappen `dokumentation` hittar ni mappar som heter `vecka-1`, `vecka-2` och `vecka-3`.

På **måndagen** vecka 1 ska ni ta en skärmdump på er initial planering, och lägga in den bilden i mappen.
På **fredagen** vecka 1 ska ni ta en skärmdump på hur planeringen ser ut i slutet på veckan, samt hålla ett retrospektiv och dokumentera den.

Vecka 2 och 3 gör ni sedan samma sak.

## Kod
Som standard i detta projekt finns följande filer:
- `index.html`: Här skriver ni er HTML-kod
- `css/style.css`: Här kan ni skriva er CSS-kod (den är inlänkad i HTML, så ändra där om ni gör något annat)

Det är fritt fram att ändra i CSS/HTML, men observera att filen behöver heta `index.html` för att publicering av projektet ska fungera.
